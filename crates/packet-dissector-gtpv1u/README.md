# packet-dissector-gtpv1u

GTPv1-U dissector for packet-dissector

This crate is part of the [`packet-dissector`](https://crates.io/crates/packet-dissector)
ecosystem. It is used automatically when you enable the `gtpv1u` feature flag
on the main crate:

```toml
[dependencies]
packet-dissector = { version = "0.1", features = ["gtpv1u"] }
```

You generally do not need to depend on this crate directly.
